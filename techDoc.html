HTML?

HTML stands for Hypertext Markup Language.

HTML is a coding language that is used in most of your favorite online webpages.

With HTML, you can:

  • create your own websites.
  • start a business.
  • display information about the real world.
    • Elements

    Elements are the building blocks for HTML code.

    html, head, and body

    These are the very basic elements needed to create a website.

    Class

    Elements can have attributes known as class

    Classes can be used to target multiple elements so you can style them all at once:

    element1 class="food", element2 class="food

    Here I have targeted two different elements with the same class, then i can edit them in CSS:

    .food { color: red;}

    Now both elements containing class food will be red

    CSS

    CSS or Cascading Style Sheets is another type of code that is used to style webpages.

    CSS can:

  • Adjust the fonts of words.
  • Change the size of images.
  • Add colors and make your page pop.
    • Styling

    CSS will help you style your webpage with much freedom.

    element { background-color: green;}

    This code would make the background color of the element green.

    element { font-style: Impact;}

    This code would make the font of the element Impact instead of the default.

    Link

    In order to use CSS with your HTML code, you have to link the CSS file to your HTML in the head element.

    rel="stylesheet" href="styles.css"

    This would be the necessary code to link your CSS to your HTML
